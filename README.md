# 😥 눈물의 코딩 파티

### 🙌 진행방식
- 일시 : 화요일 9시 디스코드
- 제출 날짜 : **월요일 자정까지!!**
- 해당 주차 **연습 문제 1 + 기본 문제 3 + 응용 문제 1** 풀이합니다. 
- **각자 이름을 딴 브랜치**에서 폴더를 만든 후 풀이합니다.
- 문제를 모두 풀이하면 **PR 과 백준**에 올립니다.
- 스터디 시간 전까지 **자신을 제외한 3명의 PR 을 리뷰**합니다.
